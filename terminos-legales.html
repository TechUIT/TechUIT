3.1 Proceso de Registro

Para usar la Plataforma, el Usuario deberá crear una cuenta proporcionando información precisa y completa (nombre, correo, teléfono y contraseña segura).

TechUIT se reserva el derecho de rechazar, suspender o cancelar cualquier cuenta por incumplimientos o actividades fraudulentas.

4. FUNCIONAMIENTO DE LA PLATAFORMA

4.1 Descripción del Servicio de Intermediación

TechUIT facilita: Perfiles verificados, conexión entre partes (proximidad/servicio), procesamiento de pagos y sistema de calificaciones.

4.2 Solicitud y Pago de Servicios

El proceso de solicitud incluye:

  • Pago de Diagnóstico Inmediato: El costo del diagnóstico se debita y queda retenido en la Plataforma.
  • Conversión de Moneda: Todos los precios se muestran en Dólares estadounidenses (USD) y se convierten a la moneda local (ej. Lempiras).
  • Presupuesto: El profesional realiza el diagnóstico y envía un presupuesto para el servicio completo.
  • Pago Final: Si el cliente acepta, autoriza el pago del resto del servicio a través de la Plataforma.

4.3 Relación entre Usuarios y Técnicos

La relación contractual para la prestación del servicio se establece **directamente entre el usuario y el técnico**. TechUIT es únicamente un facilitador tecnológico.

4.4 Geolocalización y Seguimiento

La Plataforma utiliza tecnología de geolocalización para la conexión y el rastreo en tiempo real del profesional.

5. OBLIGACIONES DE LOS USUARIOS

5.1 Obligaciones Generales

Todo Usuario se compromete a usar la Plataforma de buena fe, proporcionar información veraz y no realizar actividades ilegales o fraudulentas.
